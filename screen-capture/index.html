Screen Capture

Screen Capture Browser Extension

In order to provide screen capturing you have to build a browser extension for Google Chrome.

  • Use eyeson extension.
  • Build your own extension, checkout this blog post.

Note: Mozilla Firefox supports screen capturing without building using a browser extension.
